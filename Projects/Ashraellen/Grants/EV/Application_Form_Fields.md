# Application Form Fields — Emergent Ventures

Purpose:
Collect the exact Emergent Ventures application fields and prepare institution-specific answers.

Working rule:
Answers must be clear, direct and self-contained. The main proposal must not depend on attachments.

Known / expected fields:

```text
Name
Email
Project title
Location
Website / links
How would you describe your idea in a tweet?
What is your proposal?
What are you asking funding for?
Estimated budget
Breakdown of expenses
What mainstream consensus view do you agree with?
Any supplementary materials / links
Full-time or part-time work on the project
Anything else useful for evaluation
```

EV answer strategy:
- avoid institutional fog;
- avoid spiritual, therapeutic or motivational framing;
- show the unusual founder-driven nature of the work;
- show that the project already exists and needs concentrated time and infrastructure;
- make the budget practical, not decorative;
- make the social value visible without moralising.
