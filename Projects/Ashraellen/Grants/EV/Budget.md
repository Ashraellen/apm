# Budget — Emergent Ventures

Date prepared: 2026-06-11  
Status: working EV budget draft, not yet submitted.

Purpose:
Prepare the EV-specific requested amount and expense breakdown.

Working rule:
The budget must show what funding unlocks, not merely what money covers.

Budget logic:
Funding should create sustained working time and production infrastructure for completing, publishing, translating, archiving and publicly testing the Ashraellen project.

---

## 1. Recommended request

```text
Requested amount: USD 65,000
Period: 12 months
Use: 12-month acceleration period for Ashraellen
```

Rationale:
This amount is large enough to create real protected working time and infrastructure, but still lean enough for a founder-driven, high-upside public knowledge project.

The request should not be framed as personal support. It should be framed as time protection and production infrastructure for an already existing body of work.

---

## 2. Short budget answer for the EV form

```text
USD 65,000 for a 12-month acceleration period.
```

Alternative if the form allows a sentence:

```text
I am requesting USD 65,000 for a 12-month acceleration period covering protected working time, production equipment, software and AI tools, translation and editorial support, website/archive development, publication preparation and a small contingency.
```

## 4. Explanation by category

### 4.1. Protected working time — USD 30,000

Purpose:
Create sustained work time for writing, editing, translating, publishing, recording, archiving and systematizing Ashraellen.

This is not a personal rescue request. It protects the core production condition: time.

Use:

```text
12 months of partial living / working support allowing the author to reduce external work and concentrate on the project.
```

### 4.2. Production equipment and storage — USD 8,000

Purpose:
Replace unstable or insufficient equipment and create a reliable workstation for writing, video, audio, image production and multilingual archive work.

Possible expenses:

```text
powerful laptop / mobile workstation
external monitor
external SSD / backup storage
hub or docking accessories
basic peripherals
```

Canonical explanation:

```text
The requested equipment is not a personal convenience. It is the production infrastructure required for writing, editing, translation, video and audio production, multilingual publication, archiving and public dissemination.
```

### 4.3. Software, AI, translation, hosting and production services — USD 9,000

Purpose:
Maintain the monthly tools needed to run the project professionally.

Possible expenses:

```text
AI research and production tools
language and translation support
transcription tools
video / audio editing software
image tools
hosting, domain, backup and website services
stock / music / production assets where needed
```

### 4.4. Editing, translation, formatting and publication support — USD 10,000

Purpose:
Prepare key texts and books for wider publication and institutional readability.

Possible expenses:

```text
English / Polish language review
selected literary translation and transcreation support
editing and proofreading
book formatting
cover / publication preparation
external help for selected materials where precision matters
```

### 4.5. Website, archive and multilingual infrastructure — USD 6,000

Purpose:
Strengthen Ashraellen.com as a multilingual public research archive rather than a personal site or media page.

Possible expenses:

```text
professional pages
research and method pages
book and project pages
multilingual structure
archive organization
technical improvements
long-term preservation and backup
```

### 4.6. Contingency / payment fees / administration — USD 2,000

Purpose:
Cover transfer fees, platform fees, unexpected costs, small administrative expenses and exchange-rate friction.
