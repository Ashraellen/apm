# Support Thoughts — source register and anti-duplication map

Status: active working register  
Project: Ashraellen.com / MainSite / Public Thoughts  
Created: 2026-06-25  
Purpose: prevent duplicate support thoughts across future arcs.

## How to use this file

Before creating a new support-thought arc, check every candidate against this register.

Do not check only exact wording. Check:

1. Formula / final line.
2. Core theme.
3. Central image or metaphor.
4. Emotional mechanism.
5. Slug and title.
6. Source note / Telegram source if available.

A candidate may be a duplicate even if the wording is different.

## Status values

```text
done      — published on the website
selected  — chosen for the next arc, not yet published
candidate — candidate from Telegram or another source
skip      — rejected as not suitable
merged    — partly used inside another thought
duplicate — do not use as a separate thought
review    — requires user's decision
```
