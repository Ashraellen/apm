# Content

Use this folder for public website texts and content architecture:

- homepage text;
- section descriptions;
- Research / Public / Books pages;
- professional pages;
- Kone or institution-facing landing pages;
- page drafts;
- approved formulations;

## Rule

Keep content drafts separate from technical decisions. If a text is public-facing, it belongs here. If it explains implementation, it belongs in `../Technical/`.
