# Inbox_From_Chats

Status: raw import folder

Place here all Markdown files produced by older chats that contain site-related memory.

This folder is intentionally unsorted. It is the receiving bay.

## File naming convention

Use:

`MainSite_ChatMemory_YYYY-MM-DD_short-topic.md`

Examples:

- `MainSite_ChatMemory_2026-02-16_github-pages-language-gateway.md`
- `MainSite_ChatMemory_2026-05-20_root-index-pwa-autodetect.md`
- `MainSite_ChatMemory_2026-06-08_kone-finnish-landing.md`

## Sorting rule

Do not edit these files heavily after import. Treat them as raw evidence from old chats.

Extract stable material later into:

- `../MASTER_MainSite.md`;
- `../Decisions/`;
- `../Content/`;
- `../Languages/`;
- `../Technical/`;
- `../Grant_Positioning/`;
- `../Tasks/`.
