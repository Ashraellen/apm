# Inbox_From_Chats

Status: raw import folder

Place here Markdown files produced by older chats that contain site-related memory.

Use short filenames:

`MS_01.md`, `MS_02.md`, `MS_03.md`.

Numbering: choose the next free number in this folder. If the folder is unavailable, use `MS_XX.md` temporarily.

After saving a file, a compact chat note is enough: filename, path, commit SHA, and short summary.
