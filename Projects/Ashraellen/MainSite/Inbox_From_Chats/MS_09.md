## 2. Accepted decisions

### Mini-essays / cycles

- Mini-essays should be organized as **cycles**.
- A current cycle contains up to 9 mini-essay cards.
- New mini-essays are added to the beginning of the current cycle.
- When a new cycle starts, the previous current cycle becomes an archived completed cycle.
- The preferred terminology is **cycle / цикл**, not a purely technical file/page label.
- User approved wording that makes cycles feel like a form of internal chronology or era-counting.
- Avoid multiplying folders for every cycle.
- Preferred structure: one folder `cycles/` with numbered HTML files such as `cycle-0001.html`, `cycle-0002.html`, etc.
- `essay/cycles/index.html` functions as a table of contents / archive index for completed cycles.
- The archive page label should be **Завершённые циклы**.
- On the archive page there should be a button/link back to the current cycle.
- On the current essay page there should be a small button like **Завершённые циклы** styled like the existing **читать** buttons.

### Mini-essay styling

- Mini-essay page should show 9 cards per page / cycle.
- Do not expose internal technical phrasing like “До девяти мини-эссе. Новые срезы добавляются в начало текущего цикла.” if it feels like internal machinery.
- On archive/current pages, line-height in paragraph text should be visually closer to 1.25, not loose default spacing.
- On archive cards, avoid overly bright button text; muted button style is preferred.
- Keep archive card labels clean; for the first cycle, title text should remain simply **Первый цикл**, without explanatory description if unnecessary.

### Formulas

- Formulas are conceptually different from essays.
- If essays are **cycles**, formulas should be **lines**.
- Formulas should be styled similarly to essay cards but without a **читать** button.
- Formula pages may contain more cards than essays, e.g. 12 or 15 per page.
- User supplied and approved the first formula set:
  1. Мысль возникает сама. Свобода начинается там, где человек перестаёт становиться каждой мыслью.
  2. Власть над собой — это не власть над мыслями, а власть над вниманием.
  3. Мы не выбираем первую мысль. Но можем выбрать, кормить её вниманием или нет.
  4. Мысль, с которой борются, получает силу от самой борьбы.
  5. Осознанность начинается не с тишины в голове, а с фразы: "Я вижу, что эта мысль возникла".
  6. Осознанность без тела — это концепция, которая наблюдает саму себя.
  7. Тело говорит раньше, чем ум успевает сочинить объяснение.
  8. Тело — первая дверь к себе, но не весь дом.
  9. Человек страдает не только от тревоги, но и от требования не тревожиться.
  10. Приятное состояние ещё не освобождение. Иногда это просто хорошо настроенная биохимия.
  11. Практика становится ловушкой, когда покой превращается в нового хозяина.
  12. Даже светлое переживание нужно наблюдать, иначе оно быстро становится духовной игрушкой эго.
  13. Прошлое внутри человека существует не как факт, а как воспоминание.
  14. Человек часто страдает не от события, а от версии события, которую продолжает носить в себе.
  15. Освобождение — не состояние, которое наконец наступило. Это момент, когда старая версия себя перестала казаться единственной.

### Book page pattern

The page pattern established by **Книга Нытия / The Book of Whinesis** is now the preferred template for book pages:

- title / lead;
- **О книге** or local equivalent;
- video/audiobook section;
- video description above embedded video;
- under-video note: “Видео содержит многоязычные аудиодорожки. Язык можно изменить в настройках YouTube-плеера.” and equivalents;
- excerpt block with preview and **Читать дальше / Свернуть** behaviour;
- **Статус** block instead of purchase/download placeholders;
- quiet/muted **К списку книг** button.

### Whinesis title rule

- For most non-Russian language pages, the canonical title should remain **The Book of Whinesis**.
- Do not translate the title into local versions like *Księga Narzekania*.
- Inside the literary text, translation can remain natural; not every occurrence of ordinary “нытьё” must be mechanically replaced by Whinesis.
- Ukrainian page is an exception: user approved **Книга Нытия** as the visible title for the Ukrainian page.

  - **О книге** section;
  - **Темы романа** section;

## 6. Technical notes

- GitHub update operations sometimes stalled or failed for long HTML pages.
- Reliable workaround used and accepted by the user:
  1. fetch current file and SHA;
  2. delete old file using SHA;
  3. create new file on the same path.
- This keeps the public URL unchanged while replacing page content cleanly.
- Cache-busting query parameters like `?v=1`, `?v=2`, `?v=6` were used during visual checks.
- CSS version query strings such as `books.css?v=20260519-8` were used in rebuilt pages.
- The recurring video/audio note pattern should be kept across book pages:
  - RU: “Видео содержит многоязычные аудиодорожки. Язык можно изменить в настройках YouTube-плеера.”
  - Localized equivalents on other language pages.
- Use `iframe` YouTube embeds with `allowfullscreen` and `loading="lazy"`.
- Use square visual/promo imagery where needed for site/social presentation.
