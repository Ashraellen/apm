## 1. Summary

This chat continued the Ashraellen main website build after the first MainSite chat. The main result was a completed Public/Posts structure, a new top-level Books layer, and an updated Entry page with three main directions.

The user emphasized that the existing design canon must be preserved: no random redesigns, no reverting to older hero layouts, and no rebuilding sections already finished.

Publication numbering is global across languages: the same number means the same material in every language.

Essay URL remains `/essay/`, but the stylesheet remains `assets/mini-essay.css`. User explicitly approved keeping this mismatch because RU page works.
