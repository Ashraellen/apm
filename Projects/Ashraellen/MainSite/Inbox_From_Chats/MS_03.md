### 2.7 CSS path canon

Important rule confirmed again:

If CSS lies in `/assets/`, paths inside CSS must be relative to `/assets/`.

Correct:

```css
background-image: url("backgrounds/online-bg.jpg");
background-image: url("backgrounds/whinesis-bg.jpg");
background: url("backgrounds/books-bg.jpg") center/cover no-repeat;
```

Incorrect:

```css
url("../assets/...")
url("./assets/...")
```

Those create accidental `/assets/assets/` path bugs.

### 2.8 Cache handling

GitHub Pages / browser cache caused CSS not to update immediately. Accepted practical solution:

```html
<link rel="stylesheet" href="../../assets/books.css?v=20260217">
<link rel="stylesheet" href="../assets/style.css?v=...">
```

Use a version query string (`?v=YYYYMMDD` or similar) when CSS changes.

```js

btn.addEventListener('click', function(){
  const collapsed = ex.classList.contains('is-collapsed');
  if(collapsed){
    ex.classList.remove('is-collapsed');
    btn.textContent = 'Свернуть';
  }else{
    ex.classList.add('is-collapsed');
    btn.textContent = 'Читать дальше';
    ex.scrollIntoView({behavior:'smooth', block:'start'});
  }
});
```

## 6. Technical notes

### 6.1 `window.__BASE__`

The working base script used in pages:

```html
<script>
(function(){
  const isGitHub = location.hostname.endsWith('github.io');
  const repo = location.pathname.split('/').filter(Boolean)[0];
  window.__BASE__ = (isGitHub && repo) ? `/${repo}/` : `/`;
})();
</script>
```

This should remain canonical for GitHub Pages compatibility.

### 6.2 Relative CSS paths

For `/ru/books/index.html`:

```html
<link rel="stylesheet" href="../../assets/books.css?v=20260217">
```

For `/ru/books/you-are-already-online/index.html`:

```html
<link rel="stylesheet" href="../../../assets/books.css?v=20260217">
```

### 6.3 Image paths from JS

From HTML scripts, image paths use `base`:

```js
document.getElementById('thumbOnline').src   = base + 'assets/covers/online-thumb.jpg';
document.getElementById('thumbWhinesis').src = base + 'assets/covers/whinesis-thumb.jpg';
document.getElementById('sealImg').src       = base + 'assets/symbol.png';
```

From `assets/books.css`, use:


Then upload images into those folders.

### 8.5 `Ты уже в сети` excerpt

```text
В супермаркете он устроил дефолт. Шёл вдоль рядов, глядя строго в пол. Не читал этикетки. Не сравнивал состав. Не сопоставлял цены. Рука выхватывала первый предмет на уровне пояса и клала его в корзину без паузы.

Консервы. Бумажные полотенца. Корм для аквариумных рыбок, которых у него никогда не было. Пространство вокруг начало нервничать. Экраны над полками вспыхивали хаотично. Алгоритм предпочтений захлёбывался. Ему предлагали всё сразу: подгузники, элитный алкоголь, строительные инструменты. Реклама превратилась в белый шум. Система перегревалась, пытаясь найти смысл там, где была лишь осознанная пустота.

На кассе он выложил товары и опустил взгляд. Сканер издавал короткие сигналы, но они перестали быть отдельными. Звуки слились в одну непрерывную ноту. Система не различала объекты — он отказался различать их сам.

— С вас… — начала кассир.

Он не поднял глаз. Не из протеста. Он просто не подтвердил диалог. Пауза затянулась. Время в этом секторе загустело. Когда он всё-таки посмотрел на неё, он увидел неопределённость. Лицо было набором базовых форм. Кожа — текстурой без пор. Мимика не загрузилась. Пустой слот.

— …двадцать восемь сорок, — договорила она механически, будто звук протолкнули через забитый канал.

На улице он выронил ключи. Сначала увидел, как связка коснулась асфальта. И только потом услышал звук удара. Эхо было плоским и коротким. Мир перестал рассчитывать акустику пространства. Зачем считать отражения, если слушатель не обращает на них внимания?

Теперь он знал: подтверждение — это валюта.

Мир тратит ресурсы на рендеринг только там, где есть спрос. Где ожидают боль, радость или страх. Если спроса нет — выдаётся минимально жизнеспособная версия. Черновик.

В офисе коллега застыл в полушаге. Улыбка зависла, как зацикленная анимация. Когда Митя отвёл взгляд, коллега исчез из восприятия. Система отключила сложные тени и отражения там, где не было глаз. Мир экономил электричество реальности.

Это было не разрушение. Это было истончение. Митя понял: чтобы реальность снова стала детализированной, ему придётся начать в неё верить. Или сделать шаг дальше — туда, где больше нет ни текстур, ни правил.
```
