# Session Log — Site-wide transcreation rule

Date: 2026-06-26
Project: Ashraellen MainSite

## Purpose

Record the site-wide decision that all public-facing multilingual Ashraellen.com content must be created through transcreation, not literal translation.

## Confirmed decision

Transcreation is not a Formula-only rule.

It applies to the whole website and to every new content/page type unless a specific technical, legal or canonical-label exception is stated.

## Reason

This rule was discussed before in previous chats, but there was no shared MainSite memory at that time.

It is now recorded as a stable MainSite-wide rule so future chats should not rediscover it or ask whether it applies.

## Current instruction for future chats

For any multilingual Ashraellen.com content:

```text
1. Do not literal-translate as final publication.
2. Transcreate by preserving meaning, function, rhythm, tone, cultural naturalness and reader effect.
3. Prefer final pages that contain the transcreated text directly in the published HTML.
4. Use JSON/shared renderers only when technically justified or as a temporary working layer.
```
