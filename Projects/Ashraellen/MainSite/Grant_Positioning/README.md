# Grant_Positioning

Status: working folder

Use this folder for institutional and grant-facing positioning of the Ashraellen website.

Core framing to preserve:

- independent artistic research;
- literary-philosophical research;
- practice-based cultural research;
- multilingual public research archive;
- cultural research into meaning, language, perception, inner autonomy, public speech, and digital forms.

Avoid framing Ashraellen as:

- blog;
- YouTube channel;
- spiritual teaching;
- therapy;
- motivation;
- infobusiness;
- personal request for support.

## Related site logic

The website supports the grant strategy because it shows:

- existing public archive;
- multilingual accessibility;
- professional structure;
- literary and artistic research outputs;
- long-term practice;
- production infrastructure;
- public-facing cultural form.

## Kone note

For Kone Foundation, Finnish-facing material may be useful, but the project should remain honest about its main working languages. A Finnish landing page with a link to the full English version is currently the preferred strategy.
