# Decisions

Status: accepted decision logs

Use this folder for dated, stable decisions about the Ashraellen main website.

Each decision file should include:

- date;
- source chat or source file;
- decision;
- reason;
- status;
- affected paths/pages;
- next action;
- whether the decision is implemented, drafted, or only accepted conceptually.

## Suggested filename

`YYYY-MM-DD_short-decision-topic.md`

Example:

`2026-06-22_current_chat_site_decisions.md`

## Rule

Do not put raw chat dumps here. Raw imports belong in `../Inbox_From_Chats/` first.
