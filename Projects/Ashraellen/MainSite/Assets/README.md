# Assets

Status: working folder

Use this folder for notes about site-related assets:

- logos;
- visual identity;
- icons;
- PWA icons;
- backgrounds;
- MONOLITH images;
- page illustrations;
- downloadable PDFs;
- screenshots;
- Open Graph / social sharing images;
- favicon and manifest-related assets.

## Rule

Do not store large binary assets here unless there is a clear reason. This folder is primarily for asset decisions, lists, paths, and usage notes.

If an asset is already in the live website repository, record its exact path and purpose.
