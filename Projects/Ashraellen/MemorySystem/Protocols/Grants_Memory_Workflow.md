# Grants Memory Workflow

Status: active
Created: 2026-06-22

## Purpose

This file explains how grant-related chats should preserve useful memory in the repository.

## Main folders

Grants branch:

`Projects/Ashraellen/Grants/`

Grants master:

`Projects/Ashraellen/Grants/MASTER_Grants.md`

Raw imports:

`Projects/Ashraellen/Grants/Inbox_From_Chats/`

Session logs:

`Projects/Ashraellen/MemorySystem/Session_Logs/`

## Old grant chats

For older chats, create a raw import file in:

`Projects/Ashraellen/Grants/Inbox_From_Chats/`

Use names:

`GR_01.md`, `GR_02.md`, `GR_03.md`

Record:

- grant/foundation names;
- accepted positioning;
- sent letters;
- replies and outcomes;
- drafts and ready text fragments;
- budgets and expenses;
- deadlines and requirements;
- open tasks;
- rejected or outdated ideas.

## New grant chats

For new grant work, first read:

`Projects/Ashraellen/START_HERE.md`

`Projects/Ashraellen/MemorySystem/INDEX.md`

`Projects/Ashraellen/Grants/INDEX.md`

`Projects/Ashraellen/Grants/MASTER_Grants.md`

Then work inside the relevant institution folder.

After substantial work, create a session log in:

`Projects/Ashraellen/MemorySystem/Session_Logs/`

## Stable memory

Put stable general grant decisions into:

`Projects/Ashraellen/Grants/MASTER_Grants.md`

Keep rough imports in:

`Projects/Ashraellen/Grants/Inbox_From_Chats/`
