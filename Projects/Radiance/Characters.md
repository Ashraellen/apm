# Characters — Radiance / Сияние

This file records the early character and function map for **Radiance / Сияние**.

The project is at first-canon stage. Do not treat this map as final casting.

---

## 1. Main narrator

Working role:

```text
The campfire narrator.
```

He tells children / almost children the "real" history hidden inside northern myths.

He should not be:

- a guru;
- a lecturer;
- a wizard cliché;
- a professor in disguise;
- a clean mouthpiece for the author's philosophy.

He may be:

- strange;
- tired;
- ironic;
- calm in disturbing places;
- too ordinary to be dismissed as magical;
- too exact to be dismissed as ordinary.

His authority should not come from status.

It should come from the feeling that he has either seen something, remembered something, or stopped pretending not to know it.

---

## 2. Listener group

The listeners are likely teenagers / almost children in a summer camp.

They are not a passive audience.

They protect the series from becoming a lecture.

Possible functional roles:

### The skeptic

Asks for proof, laughs at solemnity, resists the mythic frame.

Function:

```text
prevents the narrator from becoming pompous
```

### The sensitive one

Feels the danger of the stories before understanding them.

Function:

```text
registers the unseen pressure
```

### The joker

Turns fear into jokes and jokes into accidental truth.

Function:

```text
keeps the fire human
```

### The wounded one

Hears the stories as if they are addressed personally.

Function:

```text
connects mythic law to private pain
```

### The almost-adult

Refuses fairy tales, but does not leave the fire.

Function:

```text
carries the modern reader's defensive intelligence
```

## 3. Mythic figures as functions

The mythic figures should be treated as living functional principles, not as simple fantasy characters.

### Sampo

Function:

```text
abundance / provision / Creation as Manna-machine / test of participation
```

Not merely an object.

### Väinämöinen / Вяйнямёйнен

Function:

```text
song / word / name / language as tuning of reality
```

Not merely a wise old singer.

### Ilmarinen / Ильмаринен

Function:

```text
craft / forging / technology / responsibility of form
```

Not merely a smith.

### Louhi / Лоухи

Function:

```text
threshold / guarded resource / northern power / refusal to immature appetite
```

Not simply a villain.

### Pohjola / Похьёла

Function:

```text
cold access zone / other side / place where desire is tested by maturity
```

Not simply the "evil North".

### Kullervo / Куллерво

Function:

```text
trauma / damaged script / wound mistaken for destiny
```

Not merely tragic decoration.

### Lemminkäinen / Лемминкяйнен

Function:

```text
impulse / passion / death / disassembly / return
```

Not merely a reckless hero.

### Ilmatar / Илматар

Function:

```text
breath / water / gestation / birth of world / creation without violence
```

Not merely a creation figure.

### Tapio / Тапио

Function:

```text
forest / animal order / Earth body / living mechanism / measure
```

Not merely a forest god.

---

## 4. Character rule

No one should exist only to explain philosophy.

Each figure, human or mythic, must carry pressure through action, reaction, silence, question, refusal, or recognition.

Correct test:

```text
Could the scene still work if the philosophical explanation were removed?
```

If yes, the character is alive.

If no, the character is only a lecture wearing boots.
