# Research_Notes — Radiance / Сияние

Updated: 2026-06-29

This file records research orientation for **Radiance / Сияние**.

It is not a source bibliography yet.

It is a map of directions, cautions, institutional logic and responsible external framing.

---

## 1. Main research field

Primary field:

```text
Finnish / Karelian / Kalevala-related mythopoetic material
```

Use as:

```text
a living artistic-research source for philosophical speculative fiction / mythopoetic literary research
```

Do not use as:

```text
decorative Finnish flavor for a grant application
```

The Finnish / Karelian line is the main northern spine of the cycle.

Other traditions may appear as comparative echoes, but not as equal encyclopedic material.

---

## 2. Research purpose

Radiance does **not** aim to understand, modernize or explain Kalevala in a simple contemporary lens.

Correct purpose:

```text
To investigate the human being and reality through the mythopoetic traces in which epic preserved knowledge about the human being and reality.
```

Russian:

```text
Исследовать человека и реальность через мифопоэтические следы, в которых эпос сохранил знание о человеке и реальности.
```

External formulation:

```text
The project approaches Finnish / Karelian / Kalevala-related material as a living cultural trace through which contemporary literature can investigate inner laws of abundance, language, craft, threshold, trauma, death, birth, forest, measure, responsibility and participation.
```

Do not write:

```text
We explain Kalevala for the modern reader.
We retell Kalevala in a contemporary form.
We decode the true meaning of Kalevala.
```

Better:

```text
We do not retell Kalevala.
We show the world from which such stories could have arisen.
```

## 4. Personal continuity

The author previously encountered Kalevala-related material during a period of spiritual-philosophical searching.

It was one of many sources used to seek symbolic hooks toward God, consciousness, reality, language, destiny and meaning.

Current position:

```text
The author returns to this material not as a folklorist claiming ownership, but as an artist-researcher revisiting one of the sources that entered the field of his bogoiskatelstvo.
```

External caution:

```text
Do not make this personal continuity the first institutional argument.
It may explain why the material matters to the author, but the main institutional argument is the artistic-research cycle itself.
```

## 5. Comparative pantheons

Other traditions may be used as mirrors:

- Greek;
- Indian;
- Russian / Slavic;
- Biblical;
- broader mythic abundance motifs.

Main comparative focus:

```text
How different cultures described the same inner / cosmic functions through different mythic interfaces.
```

Examples:

```text
Sampo / manna / self-setting tablecloth / cornucopia / Akshaya Patra
Väinämöinen / Logos / Vedic hymn / Russian charm / Orpheus
Ilmarinen / Hephaestus / Vishvakarman / divine smith motifs
Louhi / Baba Yaga / threshold keepers / guarded knowledge
Kullervo / wounded heroes / revenge cycles / damaged destiny
Lemminkäinen / descent and return / death and reassembly
Ilmatar / cosmic egg / waters before creation / birth without conquest
Tapio / sacred groves / forest as living body / animal teachers
```

External institutional language should translate this as:

```text
mythopoetic model
symbolic systems of transmission
comparative artistic interpretation
philosophical speculative fiction
artistic research through literature
literary artistic research
practice-based literary research
```

Do not write to institutions as if proving ancient technological events.

```text
The project explores how mythic figures and motifs can be read as living structures of human perception, responsibility, language, craft, trauma, transformation and participation.
```
