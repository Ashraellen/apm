# Research_Notes — Radiance / Сияние

This file records early research orientation for **Radiance / Сияние**.

It is not a source bibliography yet.

It is a map of directions, cautions, and future institutional logic.

---

## 1. Main research field

Primary field:

```text
Finnish / Karelian / Kalevala-related mythic material
```

Use as:

```text
a living artistic-research source for philosophical speculative fiction
```

Do not use as:

```text
decorative Finnish flavor for a grant application
```

---

## 2. Personal continuity

The author previously encountered Kalevala-related material during a period of spiritual-philosophical searching.

It was one of many sources used to seek symbolic hooks toward God, consciousness, reality, language, destiny and meaning.

Current position:

```text
The author returns to this material not as a folklorist claiming ownership, but as an artist-researcher revisiting one of the sources that entered the field of his bogoiskatelstvo.
```

---

## 3. Comparative pantheons

Other traditions may be used as mirrors:

- Greek;
- Indian;
- Russian / Slavic;
- Biblical;
- broader mythic abundance motifs.

Main comparative focus:

```text
How different cultures described the same inner / cosmic functions through different mythic interfaces.
```

Examples:

```text
Sampo / manna / self-setting tablecloth / cornucopia / Akshaya Patra
Väinämöinen / Logos / Vedic hymn / Russian charm / Orpheus
Ilmarinen / Hephaestus / Vishvakarman / divine smith motifs
Louhi / Baba Yaga / threshold keepers / guarded knowledge
```

---

## 4. Institutional language caution

Internal authorial language may use:

```text
pantheons as landing teams
Sampo as Manna-machine
Creation as the true abundance machine
```

External institutional language should translate this as:

```text
mythopoetic model
symbolic systems of transmission
comparative artistic interpretation
philosophical speculative fiction
artistic research through literature
```

Do not write to institutions as if proving ancient technological events.

Correct external stance:

```text
The project explores how mythic figures and abundance motifs can be read as living structures of human perception, responsibility, language, craft, and inner transformation.
```
