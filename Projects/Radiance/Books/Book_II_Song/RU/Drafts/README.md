Рабочая папка для русских черновиков Книги II «Песня».

- `Prologue_v1.md`, `Prologue_v2.md`, ...
- `Chapter_01_v1.md`, `Chapter_01_v2.md`, ...
- `Chapter_21_v1.md`, `Chapter_21_v2.md`, ...
- `Epilogue_v1.md`, `Epilogue_v2.md`, ...

Правило: старые версии не перезаписывать и не удалять. Каждая новая редакция создаётся отдельным файлом с новым номером версии.
