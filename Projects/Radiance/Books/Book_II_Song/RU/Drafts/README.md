# RU/Drafts — Book II: Песня

Рабочая папка для русских черновиков Книги II «Песня».

Здесь создаются только версии художественного текста.

## Правило именования

Используем формат:

`B2_Chapter_XX_vY.md`

где:

- `B2` — Book II / Книга II;
- `Chapter_XX` — номер главы двумя цифрами;
- `vY` — номер версии.

Примеры:

- `B2_Prologue_v1.md`, `B2_Prologue_v2.md`, ...
- `B2_Chapter_01_v1.md`, `B2_Chapter_01_v2.md`, ...
- `B2_Chapter_21_v1.md`, `B2_Chapter_21_v2.md`, ...
- `B2_Epilogue_v1.md`, `B2_Epilogue_v2.md`, ...

Правило: старые версии не перезаписывать и не удалять. Каждая новая редакция создаётся отдельным файлом с новым номером версии.
