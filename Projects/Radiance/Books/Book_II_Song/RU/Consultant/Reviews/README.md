# Reviews — Book II: Песня

Папка для рецензий главного консультационного чата на черновики, созданные чатами-комплектаторами.

## Правило именования рецензий

Используем формат:

- `B2_Prologue_v1_review.md`
- `B2_Chapter_01_v1_review.md`
- `B2_Chapter_01_v2_review.md`
- `B2_Epilogue_v1_review.md`

Рецензия должна содержать:

- краткий вердикт;
- что работает;
- что требует доработки;
- нарушения канона, если есть;
- проблемы объёма, ритма, персонажей или философского слоя;
- конкретную инструкцию для следующей версии;
- что обязательно сохранить при переработке.

Старые рецензии не удалять.
