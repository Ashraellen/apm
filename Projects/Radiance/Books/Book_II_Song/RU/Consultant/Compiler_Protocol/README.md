# Compiler_Protocol — Book II: Песня

Папка для постоянных инструкций и производственного протокола работы с чатами-комплектаторами.

Здесь будут храниться:

- общие правила для комплектаторов;
- правила создания файлов глав;
- правила именования версий;
- запреты и канонические ограничения;
- порядок отчёта после создания файла;
- правила взаимодействия с главным консультационным чатом.

Ключевой принцип: комплектатор не пишет полную главу в окно чата, а создаёт `.md` файл сразу в `RU/Drafts/`.
