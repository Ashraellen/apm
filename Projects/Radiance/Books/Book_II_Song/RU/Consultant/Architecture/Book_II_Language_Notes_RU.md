# Book II Language Notes — Russian usage

Project: Radiance / «Сияние»
Book: Book II — «Песня»
Status: active language note

---

## 1. «Цимбалы» — склонение

Слово **«цимбалы»** употребляется во множественном числе.

Правильный родительный падеж множественного числа:

```text
цимбал
```

Неправильно:

```text
цимбалов
```

---

## 2. Пример исправления

Неправильно:

```text
Но Марта просила тридцать секунд цимбалов для барабана-сироты.
```

Правильно:

```text
Но Марта просила тридцать секунд цимбал для барабана-сироты.
```

---

## 3. Памятка для вычитки

При вычитке глав Книги II проверять формы:

```text
нет цимбал
для цимбал
звучание цимбал
партия цимбал
тридцать секунд цимбал
```

Не использовать форму:

```text
цимбалов
```

---

## 4. Исключение

Если в живой речи персонаж намеренно ошибается или говорит просторечно, это возможно только как осознанная речевая характеристика. В авторском тексте и нормальной речи профессиональных музыкантов использовать нормативную форму **«цимбал»**.
