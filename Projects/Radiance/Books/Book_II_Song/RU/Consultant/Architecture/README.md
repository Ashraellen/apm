# Architecture — Book II: Песня

Папка для архитектурных и канонических документов русской мастер-версии Книги II «Песня».

Здесь будут храниться:

- общая архитектура романа;
- канон Книги II;
- список персонажей и их функций;
- карта глав;
- основные формулы Песни;
- ограничения по стилю, мифическому слою и линии Ивара / Айлы.

Эти документы используются главным консультационным чатом как источник канона и передаются чатам-комплектаторам как обязательный контекст.
