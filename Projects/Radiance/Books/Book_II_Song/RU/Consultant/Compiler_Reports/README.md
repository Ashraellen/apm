# Compiler_Reports — Book II: Песня

Папка для кратких отчётов чата-комплектатора после создания черновиков.

Отчёты нужны главному каноническому чату как дополнительный технический материал, но не должны вставляться в основной чат пользователя.

## Правило именования

- `B2_Prologue_v1_report.md`
- `B2_Chapter_01_v1_report.md`
- `B2_Chapter_01_v2_report.md`
- `B2_Epilogue_v1_report.md`

## Что должен содержать отчёт

- путь созданного художественного файла;
- commit SHA художественного файла;
- примерный объём в знаках;
- 5–10 пунктов содержания;
- возможные слабые места;
- что комплектатор сам считает нужным проверить главному каноническому чату.

В чат пользователю отчёт полностью не вставлять. Пользователю сообщать только путь главы, путь отчёта, commit SHA и объём.
