# Tasks_For_Compiler — Book II: Песня

Папка для полных заданий чатам-комплектаторам.

Каждый файл задания должен содержать подробную инструкцию для создания конкретного черновика главы или пролога / эпилога.

Примеры имён:

- `Prologue_task_v1.md`
- `Chapter_01_task_v1.md`
- `Chapter_01_task_v2.md`
- `Epilogue_task_v1.md`

Файл задания должен указывать:

- целевой файл в `RU/Drafts/`;
- минимальный объём;
- функцию главы;
- список сцен;
- активных персонажей;
- обязательные смысловые точки;
- запреты;
- требования к стилю, юмору и диалогам;
- финальный крючок.
