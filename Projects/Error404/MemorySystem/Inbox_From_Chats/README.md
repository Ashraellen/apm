# Inbox_From_Chats — Error404

Status: active
Created: 2026-06-22

This folder stores raw memory imports from older Error404 chats.

Use this folder before promoting stable decisions into:

`Projects/Error404/MemorySystem/MASTER_Error404.md`

## File names

Use numbered files:

`E404_CHM_001.md`, `E404_CHM_002.md`, `E404_CHM_003.md`

## What to collect

- accepted book decisions;
- chapter decisions;
- character decisions;
- canon and universe rules;
- translation rules;
- title decisions;
- file paths and commit SHAs;
- completed work;
- open tasks;
- rejected or outdated ideas;
- uncertain points needing review.

## Sorting rule

Raw chat imports stay here first.

After review, stable material may be moved into:

- `MASTER_Error404.md`;
- `Characters/`;
- `Canon/`;
- `Chapters/`;
- `Translations/`.
