# OLD_VERSIONS — Error404 / Book I / RU

This folder stores old Russian chapter versions for **Error 404: God Not Found / Book I: Why Me?**

Use this folder for previous versions that should be preserved but are no longer the current working files.

## Rules

- Current chapter files stay in `Projects/Error404/VOLUME_I/RU/`.
- Older versions move here after a newer version is accepted.
- Prefer keeping `.md` versions for archive.
- `.docx` and `.txt` old versions may be kept only when useful.
- Do not edit archived versions unless correcting a storage or metadata mistake.

## Naming pattern

Stored chapter files use compact names:

```text
E404GNFB1WM-RU_CH-XXX_vN.ext
```

Example:

```text
E404GNFB1WM-RU_CH-021_v3.md
```

Readable chapter names should be kept in indexes and notes, not necessarily in physical storage filenames.
